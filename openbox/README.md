# Install and configure Openbox on Arch Linux or Arch Linux based distros

1. Do minimal Arch installation
2. Install wget
3. wget -O - https://raw.githubusercontent.com/tscabral1/dotfiles/main/openbox/install.sh | bash
4. Reboot
