# Install and configure Openbox on Arch Linux or Arch Linux based distros

1. Do minimal Arch installation
2. Install wget
3. wget https://raw.githubusercontent.com/tscabral1/dotfiles/main/openbox/install.sh
4. bash install.sh
5. Reboot
